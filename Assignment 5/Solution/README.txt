-> To use following operations:
	Standard Deviation, Mean, Median, Mode
Seperate the numbers by using comma "," and then press the function you want to perform.

-> To use following functions:
	inverse of sine, cosine and tangent
Press shift and you will see the shift indicator, then press the corresponding function button to perform its inverse

-> Except Power, all the Scientific Funtion buttons, when pressed, return the value after the performing operation. Hence to use them, write the number first then clicl the function button.
